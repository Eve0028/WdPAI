## Description
The project is an imitation of an electronic journal. Its purpose is to display basic information about the student and his achievements, important dates, etc.
<br><br>

## Technologies
- HTML5, CSS (SCSS), JavaScript (+AJAX)
- PHP
- PostgreSQL (+PDO)
<br><br>
